# Package index

## Bootstrap branding

Utilities to apply **branding** to R plots

- [`brand_on()`](https://mbacou.github.io/mbutils/reference/brand_on.md)
  : Apply Boostrap brand to graphic devices and plots
- [`brand_off()`](https://mbacou.github.io/mbutils/reference/brand_off.md)
  : Turn Bootstrap brand off
- [`brand()`](https://mbacou.github.io/mbutils/reference/brand.md) :
  Fetch Bootstrap branding
- [`pal.brand()`](https://mbacou.github.io/mbutils/reference/pal.brand.md)
  : Bootstrap color palette
- [`brand.colors()`](https://mbacou.github.io/mbutils/reference/brand.colors.md)
  : Bootstrap color ramp
- [`thematic_brand()`](https://mbacou.github.io/mbutils/reference/thematic_brand.md)
  : Thematic-ally apply Bootstrap brand to base, lattice and ggplot2
  graphics

## Branded base R plots

Opinionated **plot** customizations

- [`axes()`](https://mbacou.github.io/mbutils/reference/axes.md) :
  Bootstrap-branded axes and labels for base R graphics
- [`par.brand()`](https://mbacou.github.io/mbutils/reference/par.brand.md)
  : Graphical parameters for Bootstrap branded plots
- [`plot_brand()`](https://mbacou.github.io/mbutils/reference/plot_brand.md)
  : Wrapper for base plot()
- [`legend_brand()`](https://mbacou.github.io/mbutils/reference/legend_brand.md)
  : Wrapper for graphics::legend()

- [`scale_brand_cf()`](https://mbacou.github.io/mbutils/reference/scale_brand_cf.md)
  : Continuous fill scale with Bootstrap colors
- [`scale_brand_dc()`](https://mbacou.github.io/mbutils/reference/scale_brand_dc.md)
  : Discrete color scale with Bootstrap colors
- [`scale_brand_df()`](https://mbacou.github.io/mbutils/reference/scale_brand_df.md)
  : Discrete fill scale with Bootstrap colors
- [`thematic_brand()`](https://mbacou.github.io/mbutils/reference/thematic_brand.md)
  : Thematic-ally apply Bootstrap brand to base, lattice and ggplot2
  graphics
- [`theme_brand()`](https://mbacou.github.io/mbutils/reference/theme_brand.md)
