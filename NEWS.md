# mbutils (development version)

### v2.3.0

* More flexible `plot_brand()` implementation

### v1.0.0

* Dropped `thematic` dependency entirely
* Renamed package from `mblabs` to `mbuils`
* Added `axes()` shorthand plotting utility for base R
* Implemented `brand_on()` and `brand_off()` to use in reports and notebooks
* Experimental `thematic_brand()` theme to use in combination with `thematic` package (but optional)

### v0.1.0

* Renamed `brand_on()` to `brand()` (returns a thematic theme)
* Fixed color definition bugs in `brand()`

### v0.0.2

* Added `ggplot2` scales and custom axes and legend to use Bootstrap branding
* Added `brand_on()` (similar to `[thematic::thematic_on()` but reads from `_brand.yml` config files instead.

### v0.0.1

* Initial commit
